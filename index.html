yui3-charrousel by tivac

A YUI3 powered carousel, because everybody should write a carousel at least once!

The Charrousel is a JS carousel with a silly name (because the Charr are awesome). It does a range of things that map tightly to exactly what I needed it to do. Adding new behaviors is pretty easy via YUI plugins.
